Quel style pour le personnage Lumo ?

Le personnage est le coeur de l'expérience — il guide, encourage et célèbre avec l'enfant. Choisis le style qui te parle.

🦊
Kawaii / Cartoon arrondi

A — Style Kawaii

Formes rondes, grands yeux expressifs, couleurs pastel. Inspiré Duolingo / Animal Crossing. Très attachant pour les petits, reste cool pour les grands.

🤖
Pixel Art / Rétro gaming

B — Style Pixel / Rétro

Esthétique jeu vidéo rétro, pixels nets, animations frame-by-frame. Ambiance Pokémon / RPG classique. Fort côté "jeu" et nostalgie.

🧙
Flat design / Moderne

C — Style Flat Moderne

Design épuré, formes géométriques, illustrations vectorielles. Inspiré Headspace / Notion. Élégant, universel, vieillit bien avec l'enfant.

✨
Créature magique / Fantasy

D — Créature Magique

Lumo comme créature fantastique lumineuse qui évolue avec l'enfant (petit -> adolescent -> majestueux). Inspiré Ghibli / Ori. Poétique et immersif.
